# Worldwide-Sentiment-Analysis-of-LLM-related-tweets
Built an end-to-end geospatial sentiment analysis pipeline on 300K+ LLM-related tweets using Python (Pandas, NLTK/VADER). Created Plotly choropleth dashboards to visualize sentiment by country/time, highlighting regions with declining sentiment to guide engagement strategies.
